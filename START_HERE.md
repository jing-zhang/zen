# 🚀 START HERE - Desktop Zen App

Welcome! This is your entry point to the Desktop Zen App.

---

## ⚡ Quick Start (30 Seconds)

```bash
cd desktop-zen-app
./run.sh          # Linux/macOS
# or
run.bat           # Windows
```

**That's it!** The app will open automatically. 🎉
